## ğŸš€ Getting Started

### Prerequisites

No installation required! This is a standalone HTML file that runs entirely in the browser.

### Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rashidrupani/math-whiz-quiz.git
   ```

2. **Open the file:**
   Simply open `MathQuizzer.html` in any modern web browser (Chrome, Firefox, Safari, Edge)

3. **Start practicing:**
   - Click "Start Quiz"
   - Read the question
   - Enter your answer
   - Submit before time runs out!
